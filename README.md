# sprint7-final-project
proyecto final del sprint 7 del curso (analista de datos) de tripleten

el objetivo del proyecto: poner en practica los conocimientos obtenidos durante el sprint 7 del curso analsita de datos, trabajando con un dataset de una empresa de telecomunicaciones
los datasets utilizados: /datasets/plans.csv
/datasets/users_latam.csv
/datasets/usage.csv
las etapas del análisis realizadas: explorar el dataset, identificacion del problema, limpieza basica, visualizacion outliers, segmentacion, insight ejecutivo y publicacion
cómo ejecutar el notebook (por ejemplo, abrirlo en Google Colab): descargar el archivo y ejecutarlo en jupiter notebook
